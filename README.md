# 🛒 Amazon Sales Dashboard | Power BI

---

## 🚀 Project Overview

This project showcases an **interactive Amazon Sales Dashboard** built using **Power BI** to analyze sales performance, customer behavior, and business insights.

The dashboard transforms raw data into **meaningful visual insights** to support **data-driven decision-making**.

---

## 🎯 Business Objectives

* 📊 Analyze overall **sales and profit performance**
* 🏆 Identify **top-performing products & categories**
* 🌍 Understand **regional sales distribution**
* 📈 Track **monthly and yearly sales trends**
* 👥 Analyze **customer segments and behavior**

---

## 📊 Dashboard Features

* 📈 **Sales Overview** (Total Sales: 1.57M)
* 💰 **Profit Analysis** (Total Profit: 175K)
* 📦 **Order Insights** (5901 Orders)
* 👥 **Customer Analysis** (773 Customers)
* 🧾 **Payment Mode Distribution**
* 🚚 **Shipping Mode Analysis**
* 🏷️ **Category-wise Sales**
* 📅 **Monthly Sales Trends**
* 📊 **Profit vs Sales Comparison (Quarterly)**
* 🔝 **Top 3 Profitable Sub-Categories**
* 🔄 **Return Status Analysis**
* 🎛️ **Interactive Filters (Region & State)**
